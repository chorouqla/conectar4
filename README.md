# Connect 4

Juego Connect 4 multijugador.

## Funciona:
✅ Login/registro
✅ Crear/unirse partida
✅ Tablero 6x7
✅ Fichas caen
✅ Turnos
✅ Victoria horizontal/vertical

## No funciona aún:
❌ Victoria diagonal
❌ Chat
❌ logout

## jugar asi:
1. Registrarse
2. Crear partida y compartir código
3. El otro se une con código
4. Click en columna para jugar

## RECURSOS UTILIZADOS

### 1. Código base del profesor (Tres en Raya)
- **Fuente:** Código proporcionado en clase (Tic-Tac-Toe / Tres en Raya)
- **Qué aproveché:** 
  - Estructura de login/registro
  - Sistema de sesiones PHP
  - Crear y unirse a partidas con código
  - Chat (adaptado)
  - Estructura de carpetas (frontend/backend)

### 2. Tutorial YouTube - "Connect 4 JavaScript Game"
- **Fuente:** YouTube / Envato Tuts+ / canal: "Web Dev Simplified" o "FreeCodeCamp"
- **Qué aprendí:**
  - Lógica del tablero 6x7
  - Gravedad (cómo caen las fichas)
  - Detección de 4 en raya (horizontal, vertical, diagonal)

### 3. Documentación oficial
- **PHP.net** - Funciones `str_split()`, `implode()`, `random_int()`
- **MDN Web Docs** - Fetch API, JSON, manipulación del DOM

### 4. Herramientas
- **XAMPP** - Servidor local (Apache + MySQL)
- **phpMyAdmin** - Gestión de base de datos
- **Visual Studio Code** - Editor de código
- **GitHub** - Control de versiones
